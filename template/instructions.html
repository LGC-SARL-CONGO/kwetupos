Product Instructions

Introduction

Welcome to the product instructions. This guide will help you install, edit, and update the template. Follow the steps below for a smooth experience.

1. Installation

Follow these steps to install the product:

  1. Download the package from the provided link.
  2. Extract the contents of the zip file to your desired location.
  3. Open a terminal and navigate to the extracted directory.
  4. Run the following command to install dependencies:
    5. npm install

2. Editing the Template

To edit the template, follow these steps:

  1. Open the project in your preferred code editor.
  2. Navigate to the src directory to find the main files.
  3. Make your desired changes in the relevant files. For example, to update the accordion component, edit src/components/BaseUI/Accordion/index.jsx or src/components/BaseUI/Accordion/Accordion.module.scss

3. Updating the Template

To update the template, follow these steps:

  1. Ensure you have the latest version of the dependencies by running:
    2. npm update
  2. Make any necessary changes to your custom files.
  3. Test the changes locally by running:
    4. npm start
  4. Once satisfied, build the project and deploy the updated model on your server.
